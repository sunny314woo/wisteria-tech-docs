## Quick Answer

ChatGPT Gemini Outline & Export is a Chrome extension that adds outline navigation, bookmarking, and Markdown export to both ChatGPT and Gemini. It solves the core problems of navigating long conversations, saving important messages, and creating permanent archives. The extension is well-designed, privacy-focused, and works seamlessly across both major AI platforms.

---

## Overview

ChatGPT Gemini Outline & Export by Wisteria Software positions itself as a conversation management toolkit for AI power users. Rather than offering dozens of unrelated features, it focuses on three core capabilities that directly address the most common pain points of heavy ChatGPT and Gemini users:

1. **Outline navigation** – See every message as a clickable entry in a sidebar
2. **Bookmarking** – Mark important messages for quick access
3. **Markdown export** – Save conversations as structured, formatted files

The extension is available on the Chrome Web Store and works on both ChatGPT and Google Gemini.

---

## Features in Detail

### Outline Navigation

When you open a ChatGPT or Gemini conversation, the extension automatically generates an outline in the sidebar. Each message appears as a clickable entry.

**How it works:** The extension analyzes the conversation DOM and extracts each message, displaying them in a tree view. User messages and assistant responses are visually distinguished.

**In practice:** Opening a 100-message conversation takes about one second for the outline to populate. Scrolling is responsive even for very long conversations. Clicking any entry jumps the conversation to that exact message.

**Performance note:** The outline does not noticeably slow page load times. It is lightweight and runs efficiently in the browser.

### Bookmarking

Each message in the outline has a bookmark icon. Click it to mark the message as important.

**How it works:** Bookmarked messages get a visual highlight in the outline. A filter option lets you show only bookmarked messages, turning a long conversation into a curated highlights list.

**In practice:** Bookmarking is genuinely useful for research workflows. If you are asking a series of questions and only a few responses contain the answers you need, bookmarking those responses creates an instant summary path.

**Limitation:** Bookmarks are stored per conversation in the browser. They persist across sessions but do not sync across devices (since they are local).

### Markdown Export

The export feature generates a clean Markdown file of your conversation.

**How it works:** Click Export, choose full conversation or bookmarked messages only, and a Markdown file downloads automatically.

**In practice:** The exported Markdown is well-structured. Headings are properly nested. Code blocks include language tags where detectable. Lists and formatting are preserved. A 50-message conversation exports in about two seconds.

**Use cases:** Exporting to Obsidian, saving to a local archive, sharing formatted conversations with colleagues, or backing up important research.

## FAQ

### Does the extension work with the latest ChatGPT interface?

Yes. The extension updates regularly to maintain compatibility with ChatGPT and Gemini interface changes.

### Can I use it without an internet connection?

The extension works within the ChatGPT and Gemini web apps, which require an internet connection. However, exported Markdown files are available offline.

### Is my data safe?

Yes. All processing is local. The extension does not transmit your conversation data to any external server.

### Does it work with Gemini Advanced?

Yes. The extension works with all Gemini tiers, including Gemini Advanced.

### Can I export only the messages I bookmarked?

Yes. This is one of the extension's standout features. You can export full conversations or only bookmarked messages.
