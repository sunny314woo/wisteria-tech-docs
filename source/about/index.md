## Wisteria Software

Wisteria Software 是一个小型的独立软件工作室，专注于开发**本地优先（local-first）**、**隐私友好（privacy-friendly）** 的技术工具。

### 产品线

- **ChatGPT Gemini Outline & Export** — 浏览器插件，用于结构化管理 ChatGPT/Gemini 对话，提供大纲视图、书签、高亮与 Markdown 导出功能。
- **Inbox** — macOS 本地优先工具，支持 Markdown / JSONL 导入，聚焦快速记录与离线可用。

### 技术方向

- Chrome Extension 开发
- FastAPI 后端
- MySQL / 原生 SQL
- 本地优先架构设计
